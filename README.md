Glass Terminal 🖥️✨
A sleek, unified mobile terminal and VNC client designed for seamless remote access to headless environments, drop-boxes, and servers. Built for developers and security professionals who need a reliable, frictionless connection to their remote architecture on the go.

🚀 Core Features
Unified Access: Integrated SSH and VNC client in a single, modern interface.

Headless-Optimized: Built specifically to handle headless Raspberry Pi deployments, Kali Linux drop-boxes, and cloud servers.

Streamlined Authentication: Securely store and manage SSH keys and VNC passwords for instant, one-tap deployment access.

Glass UI: A clean, translucent aesthetic that maximizes screen real estate for your terminal sessions and remote desktops.

Session Persistence: (Optional based on your build) Keep sessions alive in the background or seamlessly reconnect to detached tmux/screen environments.

🛠️ Use Cases
Red Team Operations: Instantly connect to headless Pi-Tail or custom Kali payloads via local hotspots or USB data links.

Homelab Management: Manage your local network nodes, Docker containers, and IoT devices without reaching for a laptop.

On-the-Go Development: Jump into a remote server environment whenever critical updates or reboots are required.

⚙️ Tech Stack
Frontend: Flutter (Adjust if using native Android/Kotlin)

Protocols: SSH-2.0, RFB (VNC)

Encryption: AES-256 for local credential storage
